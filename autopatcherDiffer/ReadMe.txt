
*Finding differences between two selected versions and combining insert,update and delete into a single file for each.(Also creating their content files.)

=>> In other words, for two different versions we will have 5 files at the end such as "insert"(all files which are inserted in new version as combined into a single file),"insertcontent"(offset of each file in insert file),
"update"(all files which are updated in new version as combined into a single file),"updatecontent"(offset of each file in update file),"deletecontent"(list of files which are deleted in new version).
